Craigslist Event Calendar Project

This group project is about redesigning the imperfect Event Calendar on the infamous Craigslist website.

Elevator Remote Application Project

This project is about designing a elevator remote control concept that is convenient for people in the COVID-19 pandemic.
We ended up desinging the elevator interior as well as extriors, moreover, we designed an app interface that works with the elevator.

Elevator Exterior

We designed the elevator exterior with foot pedals to replace the elevator calling buttons.

Elevator Interior

We designed the elevator interior with a screen that shows QR code for scanning and selecting floor via smartphone application.

Smartphone Application Storyboard

This includes all the interfaces we designed for the smartphone application.
